# Part 1 Repository
## Project Description
The program ensures that the user input is correct before allowing access to the system.

## Features
Username validation
Password complexity check
Phone number validation
User registration 
User login system

## Validation Rules 
### Username
Must be no more than 5 characters and must contain can underscore.

### Password 
Must be no more than 8 characters long also must contain a capital letter, a number and a special character.

### Phone Number
Must start with +27 and have a correct length.

## How the program works
1. Username, password and phone number entered by the user.
2. Each input validated by the system.
3. The user is registered if all their details are correct.
4. The user their credentials to login.
5. The system displays a success or failed message once it checked the login details.

## Technologies Used
Java
Object-Oriented Programming

## Classes Used
### Main Class
Controls the flow of the program
Handles user interaction

### Login Class
Stores user data
Login logic handled
Perfoms validation checks

## How to run the program
1. Use NetBeans to open the project.
2. Compile and run the main class.
3. Follow the prompts.
